# VOICEPEAK ポーズ調整ツール

VOICEPEAK の `.vpp` ファイル内に含まれる休止（pau）の長さをまとめて調整するデスクトップアプリです。Flet 製で、読点・句点だけでなく「・」「！」「？」にも対応しています。

## セットアップ

1. Python をインストールしておきます（3.8 以上を推奨）。
2. このリポジトリの`app.apy`と`setup.bat` を任意のフォルダにダウンロードします。
3. `setup.bat` を実行します。  
   - 最新版の Flet / flet-desktop をインストールします。  
   - バージョン指定したい場合は引数で渡してください（例: `setup.bat 0.29.0`）。
4. 完了後、`run_app.bat` でアプリを起動します。

## 使い方

1. 読点・句点の長さ（秒）を入力します。  
   - 読点扱い: 「、」「・」  
   - 句点扱い: 「。」「！」「？」
2. 「ファイルを選択して実行」ボタンから `.vpp` ファイルを複数選択してください（現行 Flet では OS からのドラッグ＆ドロップは未サポート）。
3. 処理後、同じフォルダに `<元ファイル名>_modified.vpp` を保存します。処理件数が表示されます。

## 主な機能

- `.vpp`（JSON）内の `pau` を指定秒数に一括変更
- 末尾の余分なデータがあっても可能な範囲で読み込み、無視した場合はメッセージで通知
- Flet UI（ライトテーマ固定）

## 注意点

- 元ファイルは上書きしませんが、念のためバックアップを取ってからご利用ください。
- OS からのファイルドラッグ＆ドロップは現在の Flet 0.28 系では非対応です。新しい Flet が対応したらアプリ側で実装を検討してください。
